This addon requires sqlite3 binary library for working.

Binary for Android and Mac come standard.  
For Linux, sqlite3 should already be installed by default in most distros.
If not, install using (debian and its derivatives) in terminal.

$ sudo aptitude install libsqlite3-0

For Windows, type the following in a J session to download and install
required dll from Jsoftware server.

load 'data/ddsqlite'
install_jddsqlite_''
